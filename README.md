# Customer Satisfaction — Seaborn Barplot

Prepared by: 25ds3000118@ds.study.iitm.ac.in

This folder contains:
- `chart.py` — Python script that generates a Seaborn barplot of average customer satisfaction by product category.
- `chart.png` — Generated PNG image (512x512 pixels).

How to reproduce:
1. Install dependencies:
   ```bash
   pip install seaborn pandas matplotlib numpy
   ```
2. Run the script:
   ```bash
   python chart.py
   ```
This will create `chart.png` in the working directory.
